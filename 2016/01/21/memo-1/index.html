  • 誰
    • 堀田ワークス　中島
    • 2015年度４月に発足した名古屋大学発の学生ベンチャーです
    • 簡単なホームページがある。まだ未完成で今後随時更新する。

      堀さんにご紹介いただきました。ありがとうございます。
  • 何をやっているのか
    • droview
      • webブラウザを使って、観光地にあるドローンを遠隔操作。
      • リアルタイムで操縦する面白い感覚　と　観光地の美しい景色　を楽しめる
      • 現在鋭意制作中で、ドローンの予約管理、課金決済システムは完成。映像配信技術を作成途中
  • 堀田ワークスの沿革としては
    • テクノロジーで新しい世界, web技術に可能性を感じた。新規性のあることをしたい。大学の仲間　さいとう　と２人で立ち上げて、現在に至る。
    • サーバーサイド、アプリケーションサイドの役割分担。
  • 強みは
    • 最新のOSSを利用した最速の開発環境
      • 従来より圧倒的な速さでサーバーの構築ができる
      • docker linuxのlxcコンテナを用いた環境　
      • AWS ライク（スケーラブルで耐障害性のある）クラウドシステム
    • アプリケーションサイド
      • UI・UXを意識でした　シングルアプリケーション製作技術
      • websocketを用いたドローンタイム制御
    • Webrtcを用いた動画配信技術
  • 最後に、本研究会の参加者の方はすごい方が多いので、見劣りする部分もありますが、
    アイデアの質ではなく量で地方活性化のたのめの新しい提案をしていきたいと思っております。
    よろしくお願いします。
